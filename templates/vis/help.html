Extending

Inputting your own data

The data for the graph is taken from a csv file that is located with the server. Therefore the data can be changed by changing the csv file itself. This can be easily dont by opening it in excell or the like. The data us read and given to the graph using JSON. The JSON data can be seen by accessing Main Graph Data. The CSV can be accessed using CSV Link
