Manual And Other Help

How to use the Application

The application is meant to display economic data about the United Kingdom.

Series Selectior


This allows for a textual selection of the series that you want to be displayed. The keyword wanted should be typed in, for example "Tax". Once, you typed in the keywords, you click "Go!". This will stelected the desired series as specificed in the search box.

Many keywords can be entered just add ", " between each of them and all the series with any of those keywords will be displayed. For example: "%, GDP, Tax, Population". All can be selected using "All"

Data Combinations


This panel, below the visualisations shows a list of interesting data and series combinations. For example, "Spending Vs. Public Debt" shows the Government spending and Public Debt series. This shows the corolation between the two and economic conclusions can be made from them.

Main Visualisation Controls


This panel controls the Main visualisation properties. There are 3 properties that can be changes:
  1. "Interactive Mode" - This toggles if the tooltip style. Either its tooltip displaying 1 point, or the tooltip shows all series in that perticular year.
  2. "Navigation Chart" - This toggles the Navigation chart that allows for the x axis to be shortened or extended.
  3. "Normalise" - This Normalises the data. This means that all the data curves will be the same height. All data will be between 0 and 1.

Main Visualisation Legend


This panel is above the main visualisations and displayes all the series information: The Series name and The Series Colour. If the circle to the left of each series is full then that means the series is being displayed, Otherwise, the circle is hollow. This means that the series is disabled and not displayed.
The legend is also a control tool. It can enable or disable series. There are two actions that can be performed:
  • Single Click - This toggles the series that is being clicked. This means it enables the series if disabled, or disables the series if enabled.
  • Double Click - This disables all other series, or displays all series. If many series are enables and a double click is performed, that series is shown only, all others are disabled. If only that one is enabled, the double click will enable all series.


Extending

Inputting your own data

The data for the graph is taken from a csv file that is located with the server. Therefore the data can be changed by changing the csv file itself. This can be easily dont by opening it in excell or the like. The data us read and given to the graph using JSON. The JSON data can be seen by accessing Main Graph Data. The CSV can be accessed using CSV Link.
