About

What is the Economics Visualisation Application?

The Economics Visualisation Application is a visualisation application that displays macroeconomic data of any country. It also provides economic theory curves and shows how the theory compares to real life historic data.
This compilation of economic data and the comparison to economic theory means that this is a great tool for teaching purposes, it provides the missing link between real life and economic theory that is often lacking in university lectures. It even provides a quiz to allow the students to check what they have or haven't learned. It can also be used by lecturers in lectures to show economic data and interesting data combinations.

Furthermore, the application provides a ecosystem of controls to allow for enhancing what can be extracted from the data into useful information. Things such as normalisation, data combinations and time range manipulation.

Technologies Used

There are many technologies used in this project. The backend and server side is built using Django. The frontend is built using pure css and bootstrap. The graphs are built using the NVD3 library which is built on D3.

Django

Django is a Content Management System for Websites. It provides web server capabilities with a managed way of storing and delivering websites to the client pc. It works on a Model-View-Template architecture. Where a view renders a template dynamically from the model data This allows for a changing (dynamic) web pages that do not have to be written fully and statically.

NVD3 and D3

These are two JavaScript Libraries. NVD3 is a visualisation library that provides a series of visualisation models e.g LineChart visualisations. NVD3 uses the D3 (Data-Driven-Documents) library to access the DOM (Document Object Model) of the webpage to insert data into it or edit it. In this case NVD3 uses D3 to access the <svg></svg> html element to draw the visualisations.

HTML and CSS

As this is a web based visualisation then html and css is required. The Page is described using Html and CSS is used for styling and positioning. Most of the website is built using pure CSS and HTML. Django renders a HTML version of the template with all the data substituted as necessary. Then D3 is used to insert the visualisations. This result is sent to the clients browser and rendered as a full web page.

The design of the website was done using mostly pure CSS as opossed to using Bootstrap framework, because bootstrap didnt display as it was supposed to and lacked the necessesary finese. Therefore, a more low level approach was taken of pure CSS to layout the page The web page is fully responsive and rescalable fitting the users screen size. (not using bootstrap).

Bootstrap

As mentioned before Bootstrap is a web development framework that allows for easily applying predefined css style to html elements. Therefore, a theme and layout options are provided that are easily applied using the class system in css. <div class="page-header"></div>. This gives the specified divide, the "page-header" css properties.
